# AI Academy

GitHub Pages-ке дайын екітілді сайт.

1. Осы папкадағы үш файлды GitHub репозиторийінің түбіріне жүктеңіз.
2. Репозиторийде **Settings → Pages** бөліміне өтіңіз.
3. **Deploy from a branch**, `main`, `/ (root)` параметрлерін таңдаңыз.

Басты бетте тек төрт бөлім көрсетіледі. Әр карточка Google Drive-тағы тиісті материалдар папкасын ашады.
